# biojs-vis-dotplot 

A module to visualize dot plots to compare protien or nucleotide sequences.

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install biojs-vis-dotplot --save
```


## Tests

```sh
npm install
npm test
```
